Apache Iceberg는 방대한 분석 데이터 세트를 위한 개방형 테이블 형식입니다. Iceberg는 SQL 테이블처럼 작동하는 고성능 테이블 형식을 사용하여 Spark, Trino, PrestoDB, Flink, Hive 및 Impala를 비롯한 컴퓨팅 엔진에 테이블을 추가합니다.

사용자 경험
Iceberg는 불쾌한 놀라움을 방지합니다. 스키마 진화 기능이 작동하며 실수로 데이터를 삭제하지 않습니다. 사용자는 빠른 쿼리를 위해 파티셔닝에 대해 알 필요가 없습니다.

스키마 진화는 추가, 삭제, 업데이트, 이름 변경을 지원하며 부작용이 없습니다.
숨겨진 파티셔닝은 사용자의 실수로 인해 조용히 잘못된 결과가 나오거나 쿼리 속도가 매우 느려지는 것을 방지합니다.
파티션 레이아웃 진화는 데이터 볼륨이나 쿼리 패턴의 변화에 따라 테이블의 레이아웃을 업데이트할 수 있습니다.
시간 여행을 통해 정확히 동일한 테이블 스냅샷을 사용하는 쿼리를 재현할 수 있으며, 사용자가 변경 사항을 쉽게 검토할 수 있습니다.
버전 롤백을 통해 사용자는 테이블을 정상 상태로 재설정하여 문제를 신속하게 해결할 수 있습니다.
